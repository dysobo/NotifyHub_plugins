🤖 AIHub 智能对话

AI 驱动的智能助手 - 对话、图片、语音、搜索

💬
智能对话
支持多种大语言模型，包括 GPT-4、Claude、Gemini、DeepSeek 等，提供流畅的对话体验。
🖼️
图片处理
图片分析：发送图片让 AI 进行内容识别和分析
图片生成：根据文字描述创建精美图片
🎤
语音功能
语音识别（STT）：将语音转换为文字
语音合成（TTS）：将文字转换为自然语音
🔍
Web 搜索
AI 可以联网搜索最新信息，获取实时数据，提供准确的回答。

📖 命令使用指南

💬 普通对话 直接发送文字消息即可与 AI 对话
🖼️ 图片分析 直接发送图片，AI 会自动分析图片内容
🎨 生成图片 /draw 描述 或 /画图 描述（例如：/draw 一只可爱的猫咪）
🔍 Web 搜索 /search 关键词 或 /搜索 关键词（例如：/search Python 教程）
🔊 语音合成 /tts 文字 或 /语音 文字（例如：/tts 你好世界）
🎤 语音识别 直接发送语音消息，AI 会自动转换为文字并处理
🔄 清空历史 /clear 或 /清空 或 /reset 或 /重置
